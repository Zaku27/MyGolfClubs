# MyGolfBag (Web)

Web app for managing golf club specs with React + TypeScript + Vite.

## Project Status

- Active development target: Web only
- Flutter code under `flutter/`: kept for reference/history, not under active development

## Quick Start

```bash
npm install
npm run dev
```

If port 5173 is already in use, Vite automatically switches to the next available port.

## Common Commands

```bash
npm run dev
npm run build
npm run lint
npm run preview
```
